# CODEX·IA — Brand Guide

## Name
Always written as **CODEX·IA** (uppercase, with punto medio Unicode U+00B7).

### Correct
- CODEX·IA
- @corpusia/codexia-mcp (npm package, lowercase allowed)
- codexia.mx (domain, lowercase)

### NEVER write
- CodexIA
- Codex IA
- codexia
- CODEX-IA
- Codex-IA

## Logo

The CODEX·IA logo consists of a balance scale icon with a digital teal (#0E6E9E) plate, book/codex ornament, and the wordmark.

- "CODEX" in charcoal (#2B2B2B) serif bold
- "·IA" in primary blue (#0E6E9E)
- The icon (balance alone) can be used as favicon or app icon without text

## Spacing
Minimum clear space around logo: 1× height of the letter "C" on all sides.

## Relationship with CORPUSIA
- CORPUSIA = parent brand (SaaS platform)
- CODEX·IA = child brand (MCP server / legal knowledge engine)
- When CODEX·IA appears alone, include "Powered by CORPUSIA"
- npm: @corpusia/codexia-mcp (links both brands)
