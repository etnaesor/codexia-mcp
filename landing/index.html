El problema con la IA y la ley mexicana

  • ✕
    Alucinaciones legales. Los LLMs inventan artículos, citan leyes inexistentes y mezclan jurisdicciones.
  • ✕
    Legislación fragmentada. 8 portales gubernamentales, formatos incompatibles, encodings distintos.
  • ✕
    Desactualización silenciosa. Reformas del DOF que cambian artículos sin aviso.
  • ✕
    Barrera técnica. Acceder a la ley mexicana requiere scrapers custom y mantenimiento constante.

CODEX·IA: la ley como herramienta

  • ✓
    Zero alucinaciones. get_article() devuelve texto verbatim de la fuente oficial.
  • ✓
    Fuente unificada. 8 fuentes normalizadas en SQLite FTS5. Búsqueda <100ms.
  • ✓
    Drift detection. check_updates() compara contra el DOF diariamente.
  • ✓
    Una línea de código. npm install y Claude consulta la ley directamente.

MCP Tools

Cuatro herramientas, toda la ley

Cada tool devuelve texto verbatim directo de la fuente oficial.

⚡ search_lawFull-text
Búsqueda inteligente

Búsqueda full-text con FTS5 snippet() optimizada para español.

Ejemplo
search_law("derechos trabajador despido")
→ 23 resultados en 28ms
§ get_articleVerbatim
Texto exacto de la ley

Texto completo y literal de un artículo. Zero LLM summarization.

Ejemplo
get_article("LFT", "48")
→ Art. 48 [texto completo]
☰ list_statutesCatálogo
Catálogo navegable

Toda la legislación indexada filtrable por materia y jurisdicción.

Ejemplo
list_statutes({ materia: "laboral" })
→ 24 leyes
↻ check_updatesDrift
Reformas en tiempo real

Compara contra el DOF. Detecta reformas y marca artículos desactualizados.

Ejemplo
check_updates({ since: "2026-01-01" })
→ 7 reformas detectadas

Documentación

Empieza en 3 pasos

Desde cero hasta consultar la Constitución en menos de 2 minutos.

PASO 01
Instala el paquete

Disponible en npm. Incluye la base SQLite pre-construida.

$ npm install @corpusia/codexia-mcp

# O directamente
$ npx @corpusia/codexia-mcp
PASO 02
Configura tu cliente MCP

Agrega el endpoint a Claude Desktop o Cursor.

"mcpServers": {
  "codexia": {
    "type": "url",
    "url": "https://codexia.corpusia.net/mcp"
  }
}
PASO 03
Consulta la ley

Pregunta en lenguaje natural. Claude invoca CODEX·IA automáticamente.

user → ¿Cuál es el plazo para
        interponer amparo directo?

⚡ search_law("plazo amparo directo")
→ Art. 17 Ley de Amparo [verbatim]
