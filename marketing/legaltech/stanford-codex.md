## Tool Name
CODEX·IA — Mexican Law MCP Server

## Organization
CORPUSIA (Mexico City)

## Description
Open-source MCP server providing structured, machine-readable access to Mexico's complete legal corpus (59,000+ documents). Uses SQLite FTS5 for full-text search and returns verbatim statutory text via Model Context Protocol. Addresses legal AI hallucination through source-grounded retrieval. Apache 2.0 license.

## Relevance to CodeX
- Computational law: first structured API for Mexican legislation
- AI + Law: novel approach to hallucination via MCP tool-use
- Access to justice: open-source infrastructure for Mexican legal information
- Cross-jurisdictional: based on Ansvar pattern (80+ countries)
