## Short Description (160 chars)

## Full Description
CODEX·IA indexes all Mexican legislation and exposes it via Model Context Protocol. Built on the Ansvar Systems pattern (proven in 80+ jurisdictions), it provides four tools: search_law (full-text FTS5), get_article (verbatim statutory text), list_statutes (browsable catalog), and check_updates (drift detection against Mexico's Official Gazette). TypeScript + SQLite FTS5 + Vercel serverless. The only structured API for Mexican law — no government endpoint exists.
